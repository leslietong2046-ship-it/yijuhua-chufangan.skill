# 一句话出方案 📋

> **你有没有花3小时憋方案框架，最后发现方向全错？**

现在只需要说一句话，5步出完整方案。

## 怎么用

1. 说一句话："帮我写个数字化转型方案"
2. Agent反问确认：给谁看？什么场景？
3. 5步闭环输出完整框架

## 5步流程

1. **理解需求** → 确认核心问题、受众、场景
2. **定位受众** → 决定语气和深度
3. **搭框架** → 背景+目标+举措+保障+成效
4. **填论点** → 每个举措用"动作+对象+效果"公式
5. **检查逻辑** → 自检5问

## 覆盖场景

- 🏛️ 政务方案（政策背景+量化指标+保障措施）
- 💼 商业方案（市场机会+SMART目标+资源需求）
- 💻 技术方案（需求分析+技术选型+风险对策）

> 一句话进，五步走，框架出来不用愁。
