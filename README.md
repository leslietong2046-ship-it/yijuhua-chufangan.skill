## 🎯 一句话说明

覆盖政务/商业/技术三大场景，动作+对象+效果公式，**领导看完不会问"然后呢"**。

---

## 💡 核心方法

### 一句话进，五步走

```
需求 → 受众 → 框架 → 论点 → 检查
```

**1. 理解需求**
- 一句话定位核心问题
- 确定方案类型（政务/商业/技术）

**2. 定位受众**
- 领导层：看成绩、看价值
- 同事层：看分工、看配合
- 执行层：看行动、看标准

**3. 搭框架**
- 政务版：背景→目标→举措→保障→成效
- 商业版：市场→痛点→方案→价值→落地
- 技术版：现状→问题→方案→实现→运维

**4. 填关键论点**
- 每个举措用**「动作+对象+效果」**公式
- ❌ "加强培训"
- ✅ "面向全区窗口人员开展实操培训，考核通过率提升至95%"

**5. 检查逻辑**
- 领导看完会问"然后呢"吗？
- 每个举措能落地吗？
- 有没有漏掉关键干系人？

---

## 🏆 核心公式

### 「动作+对象+效果」

| 常见废话 | 落地表达 |
|----------|----------|
| 加强管理 | 明确责任分工，制定考核机制 |
| 加大宣传 | 制作宣传物料，开展专题培训 |
| 优化流程 | 精简审批环节，缩短办理时限 |
