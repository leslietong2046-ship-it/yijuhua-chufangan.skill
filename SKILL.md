# 一句话出方案

> **你有没有花3小时憋一个方案框架，最后发现方向全错？**
> 
> 别慌，现在你只需要说一句话，我来帮你搞定框架。

---

## 核心方法：5步出方案（3分钟闭环）

### 第1步：理解需求（AI来做）
用户丢过来一句话后，**先反问确认3个关键**：
- 核心要解决什么问题？
- 方案给谁看？
- 什么场景用？

> **示例输入**："领导让我写个数字化转型方案"
> **确认问题**："这个方案是给上级汇报、还是内部落地执行？目标受众是局长还是技术部门？"

---

### 第2步：定位受众（决定语气和深度）

| 受众类型 | 语气风格 | 重点关注 |
|---------|---------|---------|
| 领导/上级 | 简洁、数据说话、结论先行 | 意义+成效+投入产出 |
| 同事/平级 | 平等协作、讲清利弊 | 可行性+资源需求 |
| 执行层 | 具体可操作、步骤清晰 | 时间表+责任分工+风险预案 |

---

### 第3步：搭框架（核心框架模板）

**通用框架 = 背景+目标+举措+保障+预期成效**

| 模块 | 必答问题 | 字数建议 |
|-----|---------|---------|
| 背景 | 为什么现在要做这事？ | 100-200字 |
| 目标 | 做完了变成什么样？（量化） | 50-100字 |
| 举措 | 具体干哪几件事？（3-5条） | 500-800字 |
| 保障 | 怎么确保能落地？ | 200-300字 |
| 成效 | 衡量指标是什么？ | 100-200字 |

---

### 第4步：填关键论点（每个举措的套路）

每个举措用这个公式：**动作+对象+效果**
- ❌ "加强培训" 
- ✅ "面向全区窗口人员，开展'一网通办'实操培训，考核通过率提升至95%"

---

### 第5步：检查逻辑（自检清单）

- [ ] 目标能衡量吗？
- [ ] 举措和目标对得上吗？
- [ ] 有时间节点吗？
- [ ] 有责任部门吗？
- [ ] 领导看完会问"然后呢"吗？

---

## 场景模板库

### 🏛️ 政务方案模板
```
【XX区/局数字化转型方案（征求意见稿）】

一、背景与依据
（政策背景+现状问题，引用上级文件精神）

二、总体目标
（1-2句话，量化指标：到XX年，XX率从X%提升至X%）

三、重点任务
（一）任务名称
  措施：具体动作
  预期：达成效果
（二）……

四、保障措施
  组织保障/资金保障/考核机制

五、预期成效
  直接成效+间接效益
```

### 💼 商业方案模板
```
【项目名称/产品上市方案】

一、项目背景
（市场机会+用户痛点+公司战略匹配度）

二、核心目标
（SMART原则：到Q3，获取X万用户，ARR达到X万）

三、核心策略
（差异化定位+切入路径）

四、执行计划
（里程碑+甘特图视角）

五、资源需求
（人力+预算+外部支持）

六、成功指标
（北极星指标+辅助指标）
```

### 💻 技术方案模板
```
【XX系统/平台建设方案】

一、需求分析
（现状痛点+用户故事）

二、技术选型
（架构图+选型理由）

三、建设内容
（功能模块分解）

四、实施路径
（阶段划分+交付物）

五、风险与对策

六、资源评估
（人力+服务器+第三方服务成本）
```

## 输出示例

**用户输入**："给我一个关于推广智能客服的方案"

**Agent输出**：
```
【XX单位智能客服推广方案】

一、背景
根据《国务院关于加快推进政务服务标准化规范化便利化的指导意见》，群众对7×24小时服务需求迫切，我单位当前热线人工接听率仅65%，亟需智能化升级。

二、目标
到2024年底，智能客服独立解决率达70%，人工接听率降至40%，群众等待时长压缩至30秒以内。

三、重点任务
（一）建设统一知识库
  汇聚高频问题500+条，实现跨渠道共享
（二）部署智能语音导航
  覆盖查询、预约、投诉三大类场景
（三）建立"人机协同"机制
  机器无法解决时，无缝转接人工并推送上下文

四、保障措施
成立专班（办公室+信息科+窗口代表），分两期推进……

五、预期成效
| 指标 | 当前值 | 目标值 |
|-----|-------|-------|
| 自助解决率 | 35% | 70% |
| 平均等待时长 | 3分钟 | 30秒 |
| 群众满意度 | 78分 | 90分 |
```
