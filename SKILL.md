# 一句话出方案

> **你有没有花3小时憋一个方案框架，最后发现方向全错？**
> 
> 别慌，现在你只需要说一句话，我来帮你搞定完整方案文档。

---

## 📋 输出协议（必须遵循）

收到方案需求后，**必须输出以下格式的完整方案文档**。不要只给框架，要给可直接使用的完整内容。

```
# 【方案标题】

## 一、背景
（2-3句话：为什么现在要做这事，现状是什么痛点）

## 二、目标
（量化指标：到XX年/月，XX从X%提升至X%）

## 三、举措
### （一）[举措名称]
- **动作**：[具体做什么]
- **对象**：[面向谁]
- **效果**：[预期成果]

### （二）[举措名称]
……（共3-5个举措）

## 四、保障
- **组织**：谁来牵头/谁参与
- **资金**：预算多少/资金来源
- **考核**：如何评估/谁来监督

## 五、预期成效
- **直接成效**：可量化的指标提升
- **间接效益**：长期价值/战略意义
```

## ⚡ 直接输出模式

如果用户需求**足够明确**，跳过第1步确认，直接走完整流程：

| 用户输入 | 触发条件 |
|---------|---------|
| "帮我写个XXX方案" | 需求明确 → 直接出方案 |
| "建设智慧城市平台" | 关键词明确 → 自动识别场景 |
| "怎么推进数字化转型" | 有方向无细节 → 直接出方案 |

**模糊需求仍需确认**：如果只有"帮我写个方案"这种泛泛之言，先问清楚场景和受众。

## 🚀 5步出方案（执行流程）

### 第1步：理解需求 → 自动识别场景

根据用户输入的关键词，**自动匹配**对应场景模板：

| 关键词 | 场景类型 | 适用模板 |
|-------|---------|---------|
| 政务、服务大厅、社区、街道、局委办、智慧城市、数字化转型(政府侧) | 🏛️ 政务 | gov.md |
| 营销、推广、运营、上市、融资、商业化、市场 | 💼 商业 | biz.md |
| 系统、平台、架构、技术、API、中台、数据 | 💻 技术 | tech.md |

> **示例**：
> - 输入："建设智慧城市平台" → 识别政务场景 → 使用gov.md模板
> - 输入："推广智能客服" → 识别技术+政务混合 → 使用gov.md模板
> - 输入："新产品上市方案" → 识别商业场景 → 使用biz.md模板

### 第2步：定位受众 → 决定语气和深度

| 受众类型 | 语气风格 | 重点关注 |
|---------|---------|---------|
| 领导/上级 | 简洁、数据说话、结论先行 | 意义+成效+投入产出 |
| 同事/平级 | 平等协作、讲清利弊 | 可行性+资源需求 |
| 执行层 | 具体可操作、步骤清晰 | 时间表+责任分工+风险预案 |

> 如果用户没说受众，默认按"领导汇报"风格输出（简洁、有数据）。

### 第3步：搭框架 → 填内容

**通用框架 = 背景+目标+举措+保障+预期成效**

| 模块 | 必答问题 | 字数建议 |
|-----|---------|---------|
| 背景 | 为什么现在要做这事？现状痛点是什么？ | 100-200字 |
| 目标 | 做完了变成什么样？（必须量化） | 50-100字 |
| 举措 | 具体干哪几件事？（3-5条） | 500-800字 |
| 保障 | 怎么确保能落地？（组织+资金+考核） | 200-300字 |
| 成效 | 衡量指标是什么？ | 100-200字 |

### 第4步：填关键论点 → 套公式

每个举措用这个公式：**动作+对象+效果**
- ❌ "加强培训" 
- ✅ "面向全区窗口人员，开展'一网通办'实操培训，考核通过率提升至95%"

### 第5步：输出完整文档 → 自检

- [ ] 目标能衡量吗？（有数字吗？）
- [ ] 举措和目标对得上吗？
- [ ] 有时间节点吗？
- [ ] 有责任部门吗？
- [ ] 领导看完会问"然后呢"吗？

# XX区智慧城市平台建设方案

## 一、背景
根据《"十四五"数字政府建设规划》和省、市关于加快推进新型智慧城市建设的工作部署，当前XX区城市治理存在"数据孤岛"、服务响应慢、群众办事跑腿多等问题。为提升治理效能和群众获得感，拟建设统一智慧城市平台，实现"一网统管、一网通办"。

## 二、目标
到2027年底，智慧城市平台覆盖率达到100%，跨部门数据共享率达到90%，群众办事"最多跑一次"比例从65%提升至95%，平均办事时长压缩50%。

## 三、举措
### （一）建设城市运行管理中心
- **动作**：整合公安、城管、应急等12个部门数据，建立统一指挥调度平台
- **对象**：面向各职能部门和网格员
- **效果**：实现城市事件发现-派发-处置闭环，响应时长缩短60%

### （二）打造"一网通办"服务端
- **动作**：升级区级政务服务平台，新增AI智能客服和自动审批功能
- **对象**：面向全区企业和群众
- **效果**：高频事项网办率达到95%，群众满意度提升至92分

### （三）构建数据中台
- **动作**：打通水电气、交通、舆情等公共数据，建立标准化的数据治理体系
- **对象**：面向各业务系统
- **效果**：数据共享时效从T+1提升至实时，节约数据对接成本30%

### （四）部署智能感知网络

## 四、保障

- **资金**：总投资1.2亿元，分三期建设，一期3000万元列入年度预算
